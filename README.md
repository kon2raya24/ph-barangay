# ph-barangay

[![npm version](https://img.shields.io/npm/v/ph-barangay.svg)](https://www.npmjs.com/package/ph-barangay)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

Philippine barangay code utilities and validation

## Installation

```bash
npm install ph-barangay
```

## Functions

- `validateBarangayCode` — Validate 10-digit PSGC barangay code
- `getProvinceFromCode` — Extract province code from PSGC
- `getCityMunicipalityFromCode` — Extract city/municipality code from PSGC
